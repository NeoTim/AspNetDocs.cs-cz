# <a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>Postupy: vrácení dat ve formátu JSON pro volání AJAX ve webové aplikaci ASP.NET MVC?

autor – [Chris pixelů na](https://twitter.com/chrispels)

V tomto videu Chris pixelů na ukazuje, jak pomocí tříd JsonResult a JSON vracet instance tříd jako data ve formátu JSON. Nejdřív se vytvoří Ukázková webová aplikace MVC a přidají se kontroler a přidružené zobrazení pro informace o zákaznících. Pak je v rámci datového modelu vytvořena jednoduchá třída zákazníka. Funkce GetCustomer () se pak přidá k řadiči zákazníka, aby načetla data pro jednoho zákazníka ve formátu JSON. Nakonec se skript přidá do zobrazení zákazníka, které používá jQuery pro volání metody GetCustomer (), aby se načetla zákaznická data ve formátu JSON.
