<a name="code-coverage-of-automated-tests"></a>Pokrytí kódu automatickými testy
====================
podle Eric Lee

Toto video ukazuje, jak použít pokrytí kódu jako metriku pro určení, jak efektivní naše testování je z webové aplikace ASP.NET. Po dokončení testu Visual Studio Team System hlásí procentuální podíl kódu, který se provedl a označuje spuštěný kód s zelené na pozadí a odvolat kód se červené na pozadí, usnadňuje zobrazíte kód, který se vztahuje test. Dozvíte se taky, jak pomocí jazyka C# nebo Visual Basic k rozšíření automatizovaného testu.
