# <a name="getting-started-with-reporting-services"></a>Začínáme se službami Reporting Services

podle [společnosti Microsoft](https://github.com/microsoft)

Služba SQL Server Reporting Services umožňuje navrhovat a nasazovat pěkně formátované a interaktivní sestavy představující vaše data. Vhodné pro tisk nebo on-line distribuci, tyto zprávy a umožňuje získat profesionální výsledky velmi rychle. Toto video ukazuje, jak nastavit Reporting Services a Business Intelligence Development Studio vytvářet sestavy pomocí průvodců a návrhářů a nasadit je na server sestav.
