# <a name="jquery-cycle-294"></a>jQuery Cycle 2.94

Následující soubory cyklů jQuery jsou hostovány v této síti CDN:

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js) &mdash; plnou verzi modulu plug-in pro cyklus jQuery. Zahrnuje vše více než dvě desítky s dalšími možnostmi a všechny možnosti.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js) &mdash; verzi minifikovaného souboru. Tato verze se používá pro produkční aplikace ke zlepšení výkonu vašeho webu.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js) &mdash; Tato verze modulu plug-in cyklu jQuery je menší a obsahuje pouze efekt přechodu "slábnutí".
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js) &mdash; verzi minifikovaného souboru. Tato verze se používá pro produkční aplikace ke zlepšení výkonu vašeho webu.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js) &mdash; ještě menší verzi modulu plug-in pro cyklus jQuery, který obsahuje pouze přechod "zeslabit" bez všech možností.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js) &mdash; verzi minifikovaného souboru. Tato verze se používá pro produkční aplikace ke zlepšení výkonu vašeho webu.
